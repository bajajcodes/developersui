Default Buttons

Default style for simple button, full-width button and disabled button.

Solid Buttons

Solid style buttons comes in four different options danger, primary, secondary and success. On hover each button background color changes.

Outlined Buttons

Buttons with white background, colored border and colored font as per button type danger, primrary, secondary and success.

Small and Large Button

Buttons in three different sizes options default, large and small.

Rounded Button

Add shape round to any of button types default, outline and solid.
