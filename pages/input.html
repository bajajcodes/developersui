Input Text

Input Texbox

Input text comes in three sizes small(sm), medium (md and default) and large(lg). Input default style is associated with blue color on different actions and properties.

Input Correct and Incorrect

Correct input is associated with green color which also mean success, whereas Incorrect input is associated with red color which also mean danger. Rest default input styles are preserved.
